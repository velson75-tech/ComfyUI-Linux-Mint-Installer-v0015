Установка ComfyUI на чистую Linux Mint (2024–2026)

Версия инструкции: январь 2026
Подходит для Linux Mint 21.x / 22.x (Cinnamon, XFCE, MATE)

Предварительные требования
──────────────────────────
• Свободное место на диске: минимум 20–40 ГБ
• Оперативная память: минимум 8 ГБ (лучше 16+ ГБ)
• Интернет-соединение стабильное
• Установлена свежая Linux Mint (желательно с обновлениями)
• Если есть видеокарта NVIDIA — драйверы уже должны стоять (или установятся автоматически)

Шаг 1. Обновление системы и установка нужных пакетов
─────────────────────────────────────────────────────
1. Откройте терминал (Ctrl + Alt + T)
2. Выполните по очереди (вводите пароль, когда попросит):

   sudo apt update
   sudo apt upgrade -y
   sudo apt install -y python3 python3-venv python3-pip git wget curl gnome-terminal

Шаг 2. Скачивание установщика ComfyUI
───────────────────────────────────────
В том же терминале выполните:

   cd ~/Рабочий\ стол    # или cd ~/Desktop, если у вас английская локаль
   wget https://raw.githubusercontent.com/ваш-ник-или-ссылка/main/comfyui_installer.py -O comfyui_installer.py

   # Если файла нет в репозитории — просто сохраните код установщика вручную в файл comfyui_installer.py

   chmod +x comfyui_installer.py

Шаг 3. Запуск графического установщика
────────────────────────────────────────
В терминале выполните:

   ./comfyui_installer.py

Откроется окно программы.

Что делать в окне:
1. В выпадающем списке «Профиль видеокарты» оставьте «Auto detect» или выберите подходящий вариант.
2. Нажмите большую кнопку «Установить / Обновить».
3. Дождитесь завершения (5–40 минут, зависит от интернета и железа).
   - Прогресс-бар дойдёт до 100%
   - В логе появится «=== УСПЕХ ===»
   - Через 0.5–1 секунду откроется большое окно-сообщение:

     • Ярлык добавлен в меню «Графика»
     • При запуске откроется браузер
     • При запуске откроется терминал
     • Дождитесь окончания загрузки ComfyUI
     • Обновите вкладку

   Нажмите OK.

Шаг 4. Первый запуск ComfyUI
─────────────────────────────
1. Откройте меню → Графика → найдите «ComfyUI» (или наберите в поиске «ComfyUI»)
2. Запустите ярлык

Что произойдёт:
- Откроется терминал с большим текстом-предупреждением
- Почти сразу откроется вкладка браузера[](http://127.0.0.1:8188)
- В терминале начнётся загрузка моделей, нод и т.д. (при первом запуске 1–10 минут)

Важно при первом запуске:
- Если вкладка чёрная / пустая / ошибка 404 → просто обновите страницу (F5 или кнопка обновления)
- НЕ закрывайте терминал вручную, пока работаете с ComfyUI
- Когда закончите → в терминале нажмите Ctrl+C (один раз), затем Enter

Шаг 5. Что делать после установки
──────────────────────────────────
• ComfyUI полностью готов к работе
• Интерфейс — в браузере по адресу http://127.0.0.1:8188
• Куда класть модели (checkpoints, lora, vae и т.д.):
  ~/ComfyUI-Desktop/ComfyUI/models/
• Custom nodes ставятся в:
  ~/ComfyUI-Desktop/ComfyUI/custom_nodes/
• Manager (если нужен) — ставится через терминал или вручную

Полезные команды (если что-то сломалось)
─────────────────────────────────────────
• Убить зависший ComfyUI (если порт 8188 занят):
  sudo fuser -k 8188/tcp

• Перезапустить установщик:
  cd ~/Рабочий\ стол
  ./comfyui_installer.py

• Удалить всё полностью:
  rm -rf ~/ComfyUI-Desktop
  rm ~/.local/share/applications/comfyui.desktop

Удачи с генерацией изображений!
Если будут ошибки — копируйте текст из терминала и присылайте.
